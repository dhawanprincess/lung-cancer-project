# 🫁 Lung Cancer Detection System

An AI-powered web application for early lung cancer prediction using **Deep Learning**, **Machine Learning**, and **medical imaging**. The system analyzes CT scan images along with patient clinical information to assist in the early detection of lung cancer.

> **Disclaimer:** This project is developed for educational and research purposes only. It is **not intended to replace professional medical diagnosis or clinical decision-making.**

---

## 📌 Project Overview

Early detection of lung cancer plays a crucial role in improving survival rates. This project combines medical image analysis with clinical data to build an intelligent prediction system.

The application provides an easy-to-use interface where users can upload CT scan images, provide patient information, and receive AI-generated predictions.

## 🧠 AI Architecture

The project follows a hybrid AI approach:

### Image Analysis

* Convolutional Neural Network (CNN)
* CT Scan Image Processing
* Automatic Feature Extraction

### Clinical Data Analysis

* Random Forest Classifier
* Patient Clinical Parameters
* Risk Prediction

The outputs are combined to improve overall prediction capability.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/dhawanprincess/lung-cancer-project.git
```

Move into the project directory:

```bash
cd lung-cancer-project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## ⚙️ Workflow

1. Upload a lung CT scan image.
2. Enter patient information.
3. The image is preprocessed.
4. CNN extracts image features.
5. Random Forest analyzes patient clinical data.
6. The system generates the prediction.
7. The result is displayed through the Streamlit interface.
